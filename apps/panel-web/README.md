# panel-web

Приватная админ-панель (`panel.domain.com`). Next.js + TypeScript + Tailwind +
shadcn/ui + TanStack Query/Table + ECharts. Realtime — SSE. Тёмная тема, бирюзовые
акценты.

Не имеет отношения к пользовательскому Custom CSS status-web (изоляция обязательна).
Дизайн и палитра — [../../docs/status-pages.md](../../docs/status-pages.md).
